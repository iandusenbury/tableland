# Admin page

## Skeleton features
* Title/header
* Table of dummy users
  * name
  * LinkedIn ID
  * Fake creation date
  * View profile button
  * "Positive" action button (ex: Approve)
  * "Negative" action button (ex: Deny)
* Two TEST buttons that switch between the admin and non admin views
